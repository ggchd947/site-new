Logic Gates

Gate Logic

  • A technique for implementing Boolean functions using logic gates
  • Logic gates:
    • Elementary (Nand,And,Or,Not,…)
    • Composite (Mux,Adder, …)

Elementary logic gates: NAND

A NAND gate

Functional Specification:

if (a == 1 and b == 1):
    out=0
else
    out=1
